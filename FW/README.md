# 🚀 FloatWaunch Elite v2.3
**By fw**

一款极其顺滑的网页悬浮启动器。打破浏览器与本地系统的边界，实现“网页即桌面”的极客体验。

### ✨ 核心亮点
- **Q + E 极速唤起**：非输入框状态下连按 Q, E 即可闪现，无需 Ctrl 组合键。
- **全能启动**：自动识别本地文件路径 (C:\...) 与 网页链接 (https://...)。
- **Shadow DOM 隔离**：物理级样式隔离，完美兼容 LimeStart 等复杂动态背景网页。
- **一键部署**：自带自动化安装脚本，自动配置 Windows 注册表协议。

### 🛠️ 安装步骤
1. **克隆/下载仓库**。
2. **运行安装**：双击 `setup_fw.py`，选择安装目录，脚本将自动配置 `runapp://` 协议环境。
3. **加载插件**：
   - 打开 Chrome `chrome://extensions/`
   - 开启 **开发者模式**
   - 点击 **加载已解压的扩展程序**，选择仓库中的 `extension` 文件夹。

### ⌨️ 快捷键说明
- **唤起/隐藏**：`Q` + `E` (序列按键)
- **添加项目**：直接在输入框 Ctrl + V 粘贴路径。
- **编辑路径**：对图标右键即可进入编辑模式。
